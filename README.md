## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/danyalahmad0786-spec/My-Portfolio.git
```

Navigate to the project folder:

```bash
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
